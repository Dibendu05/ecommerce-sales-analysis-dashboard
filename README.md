# E-Commerce Sales Analysis Dashboard

## Overview

This internship project presents an interactive E-Commerce Sales Analysis Dashboard developed using Microsoft Power BI.

The dashboard analyzes over 55,000 sales records and provides insights into sales performance, customer demographics, product categories, locations, and purchase methods.

## Tools Used

* Microsoft Power BI
* DAX (Data Analysis Expressions)
* Power Query
* CSV Dataset

## Dashboard Features

* Total Sales Analysis
* Gross Revenue Analysis
* Discount Analysis
* Product Category Performance
* Location-wise Sales Analysis
* Gender-wise Sales Analysis
* Age Group Analysis
* Purchase Method Analysis
* Monthly and Yearly Sales Trends

## Key Insights

* Electronics generated the highest sales.
* Credit Card was the most used purchase method.
* The 25–45 age group contributed the highest revenue.
* Mumbai and Delhi showed the strongest sales performance.
* Sales reached their peak in 2023.
